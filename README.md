StickyNotes-Local
=================

Sticky-Note App using local storage

## Install

To install, clone this repo and run the demo using your web-browser.

```bash
# git clone https://github.com/DaveTechTest/StickyNotes-Local.git
# cd StickyNotes-Local
# firefox index.html
```
